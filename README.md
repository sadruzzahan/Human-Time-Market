# Human Time Market

`human-time-market` front-end artifact exploring time-as-a-commodity marketplace UX with shared `api-server` + sandbox.

## 💼 Real-World Use Case
Founders testing scheduling / office-hour market metaphors where buyers and sellers trade bounded time slots instead of goods.

## 💰 Potential Value
- Could inform a niche skills marketplace or office-hours product if paired with payments, dispute resolution, and calendar integrations.
- Shows routing + API integration consistency with the rest of the workspace family.

## 🚀 Demo / Access
Self-hosted / local only. Bring Postgres + `.env` for `artifacts/api-server`. No hosted demo linked here.
